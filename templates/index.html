Here are some of my projects.

mystic encryption

A flask application that encrypts any message you put into it with a unique key that is generated when you create your account. It uses SQL to create queries that allow you to register an account, login, and obtain your encryption key.

mystic inventory manager

A flask application that stores inventory data for any product you would want to keep track of. Each account gets one inventory table created when you register. it uses sql queries to pull information from the table and also store new information that is input by the user.

Mystic Stock Checker

A work in progress stock checker that will check the stock of graphics cards on the best buy website. When it discoveres that a card is in stock it will send a notification to a discord server. It also has the capability to send an email notification to an email list which you can add yourself to once you are in the server.
